--- title: Clearing & Resetting layout: page pageOrder: 8 section: 'Mock Server' subsection: true ---

To clear all expectations (without restarting) use reset as follows:

Java

new MockServerClient("localhost", 1080).reset()

JavaScript

mockServerClient("localhost", 1080).reset();

Ruby

client = MockServerClient.new('localhost', 1080)
client.reset()

To clear only specific expectations use clear as follows:

Java

new MockServerClient("localhost", 1080).clear(
        request()
                .withMethod("GET")
                .withPath("/somePath")
);

JavaScript

mockServerClient("localhost", 1080).clear('/somePath');

Ruby

client = MockServerClient.new('localhost', 1080)
client.clear(request(:GET, '/somePath'))
